# Web3 2026: вертикалі (DeFi, DAO, NFT, Metaverse, Games)

**Навігація:** [[WEB3-2026-OVERVIEW]] · [[WEB3-2026-TECH-STACK]] · [[WEB3-2026-GLOBAL-EU-UA]]

---

## 0) Супершвидка карта

```
Вертикалі
├─ DeFi → фінансові протоколи без банків
├─ DAO → спільноти з ончейн-управлінням
├─ NFT → право власності на цифрові активи
├─ Metaverse → спільні 3D-світи + економіка
└─ Games → Play-to-Own (гра = власність)
```

---

## 1) DeFi (після зими)

**Стан:** відновлення + інтеграція з традиційними фінансами.

**Ланцюг перетворення:**
```
DeFi бум → криза/зломи → аудит/стандарти → інституційні кейси
```

**Ключові напрямки:**
- DEX (Uniswap та аналоги) як базова фінансова інфраструктура.
- Liquid staking (stETH та інші) — найбільший сектор.
- RWA (tokenized bonds/treasuries) — мост між TradFi та DeFi.

---

## 2) DAO (організаційний шар Web3)

**Ціль:** керування спільнотами через смарт-контракти.

**Дерево DAO:**
```
DAO
├─ Протокольні (DeFi/L1)
├─ Інвестиційні (VC/фонди)
├─ Соціальні/креативні
└─ Ігрові/гільдії
```

**Слабкі місця:**
- Низька явка в голосуваннях.
- Концентрація голосів у великих холдерів.

**Рішення:** делегати, багатопідписи, off-chain голосування.

---

## 3) NFT (з хайпу в інфраструктуру)

**Стан:** ринок охолов, але практики закріпилися.

**Використання:**
- Мистецтво (аукціони, музеї, “фіджитал”).
- Ігрові предмети та персонажі.
- Soulbound tokens для дипломів/сертифікатів.

---

## 4) Metaverse (формується)

**Позиція:** повноцінна метареальність ще не зібрана, але компоненти дозрівають.

**ASCII-ланцюг:**
```
VR/AR прогрес → 3D-світи → NFT власність → DAO управління → метаекономіка
```

**Ключове:** Web3 дає власність і переносимість активів між світами.

---

## 5) Games (Play-to-Own)

**Зміна парадигми:**
```
Play-to-Earn (спекуляції)
→ Play-and-Earn
→ Play-to-Own (власність + геймплей)
```

**Практика:**
- F2P-модель без стартових NFT.
- Невеликі, але реальні бонуси за активність.
- Ризики волатильності токенів і закриття проектів.

---

## 6) Взаємозв’язки між вертикалями (graph)

```
[NFT] ---активи---> [Games]
  |                  |
  |                  |---> [Metaverse]
  |                          |
  |                          |---> [DAO]
  |                                   |
  |---> [DeFi] <---ліквідність---------/
```

---

## 7) Практичне резюме

- **DeFi** — нова фінансова інфраструктура.
- **DAO** — новий спосіб організації.
- **NFT** — нова логіка власності.
- **Metaverse + Games** — нові економіки та досвіди.
