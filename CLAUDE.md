## Repository Type

**This is NOT a traditional software project.** It's a knowledge archive, emotional data storage system, and worldbuilding repository. There are no build tools, dependencies, or deployment pipelines. Primary formats: Markdown + JSON.

## Core Philosophy

- **Data-as-documentation:** Every file is both data and documentation
- **Preservation over deletion:** Version files, don't delete them
- **Structure precedes content:** Create containers before filling them
- **Collect, don't create:** Franko is an editor/curator, not an author
